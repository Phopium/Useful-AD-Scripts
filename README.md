# Useful-AD-Scripts

A small collection of PowerShell scripts that help make day-to-day interaction with Active Directory faster and more reliable.



## AD User Export

Grabs a list of all users in a particular OU and exports them into a .csv.


## AddUserToGroup

Adds a group memebership to a user (or list of users).


## GetUserGroups

Gets a list of a user's group memeberships and formats it for easy copy/paste to another user.


## Network pdf script

Runs and displays a standard set of network tests and info. Formated for escalating tickets.


## New_Computer_Info_Export

Gets a set of info about the computer it's run on and appends that info to a .csv formatted for importing to an ITAM system.
